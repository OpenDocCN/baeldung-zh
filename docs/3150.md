# 用 Java 生成一个安全的随机密码

> 原文：<https://web.archive.org/web/20220930061024/https://www.baeldung.com/java-generate-secure-password>

## **1。简介**

在本教程中，我们将研究在 Java 中生成安全随机密码的各种方法。

在我们的示例中，我们将生成十个字符的密码，每个密码至少包含两个小写字符、两个大写字符、两个数字和两个特殊字符。
