## **1。简介**

UTF-8 是 web 应用程序中最常用的字符编码。它支持目前世界上使用的所有语言，包括中文、朝鲜语和日语。

在本文中，我们演示了确保 Tomcat 中 UTF-8 所需的所有配置。
