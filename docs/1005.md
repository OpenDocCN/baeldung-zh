## 3.使用不同的名称公开 GraphQL 字段

客户端应用程序可能需要使用字段`first_author.`，现在它正在使用 `author`。为了满足这一要求，我们有两种解决方案:

*   更改 GraphQL 服务器中模式的定义
*   利用 GraphQL 中的别名概念

让我们一个一个来看。

这些问题与 GraphQL 提供的灵活性相矛盾。
