## 1.概观

Java 在 1.5 版本中引入了 [`enum`](/web/20221115043639/https://www.baeldung.com/a-guide-to-java-enums) 。将常量定义为`enum`使得代码可读性更好。此外，它允许编译时检查。

在这个快速教程中，让我们探索如何用一个`enum`类型的所有实例获得一个`List`。
