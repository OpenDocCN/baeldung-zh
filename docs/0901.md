## **7。新请求映射快捷方式**

Spring Framework 4.3 引入了[几个新的](/web/20221130175138/https://www.baeldung.com/spring-new-requestmapping-shortcuts) HTTP 映射注释，全部基于`@RequestMapping` :

*   `@GetMapping`
*   `@PostMapping`
*   `@PutMapping`
*   `@DeleteMapping`
*   `@PatchMapping`

这些新注释可以提高代码的可读性，减少代码的冗长。
