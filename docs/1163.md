## **1。概述**

在本教程中，我们将演示如何在 Jackson 中使用`@JsonFormat`。

`@JsonFormat`是一个 Jackson 注释，我们用它来指定如何格式化 JSON 输出的字段和/或属性。

具体来说，这个注释允许我们指定如何根据一个`SimpleDateFormat`格式来格式化`Date`和`Calendar`值。
