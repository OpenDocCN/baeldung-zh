## **1。Spring 和 Java**

#### **[> >日蚀舰第十期年度发布列车](https://web.archive.org/web/20220120072839/https://www.eclipse.org/org/press-release/20150624_mars_release.php) (+ [InfoQ coverage](https://web.archive.org/web/20220120072839/http://www.infoq.com/news/2015/06/eclipse-mars) )**

《月食火星》(4.5)已经发布了—[下面是这个版本的酷之处](https://web.archive.org/web/20220120072839/https://www.eclipse.org/eclipse/news/4.5/jdt.php)。此外，这是本次发布的[十大功能](https://web.archive.org/web/20220120072839/http://eclipsesource.com/blogs/2015/06/24/top-10-eclipse-mars-features/)的概要。

#### **[> >构建微服务:使用 API 网关](https://web.archive.org/web/20220120072839/https://nginx.com/blog/building-microservices-using-an-api-gateway/)**

关于在以微服务为中心的架构中使用 API 网关模式的必读文章。

根据我自己的经验，我发现代理/网关模式在巩固和明确识别系统中的职责方面做得很好。它还解决了一系列问题，如身份验证、CORS、服务发现以及受微服务早期内部重构影响的客户端。

总的来说，这是一本非常好的读物。

#### **[> >编写下载服务器。第一部分:总是流，从不完全保留在内存中](https://web.archive.org/web/20220120072839/http://www.nurkiewicz.com/2015/06/writing-download-server-part-i-always.html)**

**良好的缓存对性能良好的服务器实现大有帮助**。

#### **[> >蓝绿调配单数据库](https://web.archive.org/web/20220120072839/http://techblog.bozho.net/blue-green-deployment/)**

如果您已经将软件部署到生产环境中有一段时间了，那么您可能正在执行这个过程的某个版本。虽然是第一次设置，但感觉很好🙂

#### **[>>](https://web.archive.org/web/20220120072839/https://www.thoughtworks.com/insights/blog/5-tips-being-effective-tech-lead)**成为高效技术主管的 5 个技巧

一篇关于戴耳机屏蔽设计不当的办公室的噪音的快速而有趣的文章。

#### **[> >为什么办公室是工作走向死亡的地方](https://web.archive.org/web/20220120072839/http://swizec.com/blog/why-offices-are-where-work-goes-to-die/swizec/6695)**

这种工作方式与我和我的个性非常契合。总的来说，它当然高度依赖于一堆内部和外部因素，它可能并不适合所有人。
