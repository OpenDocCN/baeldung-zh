## 2.序列化和反序列化

[序列化](/web/20221208143921/https://www.baeldung.com/java-serialization)是**将对象的状态转换成字节流**的过程。序列化对象主要用于 Hibernate、RMI、JPA、EJB 和 JMS 技术中的。

转换方向，反序列化是使用字节流在内存中重新创建实际 Java 对象的反向过程。 这个过程常用来持久化对象。
