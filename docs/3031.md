## **2。Java**

#### **[> >用@Rule 测试 JUnit 中的文件和目录](https://web.archive.org/web/20220121012752/http://blog.codeleak.pl/2015/01/testing-with-files-and-directories-in.html)**

我不知道**这个很酷的 JUnit `@Rule`可以智能地处理文件系统**——看起来很有帮助。

#### **[>>Java 持久性锁定入门](https://web.archive.org/web/20220121012752/http://vladmihalcea.com/2015/01/12/a-beginners-guide-to-java-persistence-locking/)**

#### **[>Java 8 Streams API as Friendly ForkJoinPool Facade](https://web.archive.org/web/20220121012752/http://squirrel.pl/blog/2015/01/13/java-8-streams-api-as-friendly-forkjoinpool-facade/)**

#### **[> >关于默认方法你需要知道的一切](https://web.archive.org/web/20220121012752/http://blog.codefx.org/jdk/everything-about-default-methods/)**
