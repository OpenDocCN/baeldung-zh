因此，我们创建了四个客户，这很好，**但是请注意，他们都不在一个批处理中。**

**原因是在某些情况下，默认情况下批处理是不打开的。**

在我们的例子中，这是因为我们使用了 id 自动生成。因此，**默认情况下，`saveAll` 分别插入。**

所以，让我们打开它:

第一个属性告诉 Hibernate 以四个为一批收集插入。属性告诉 Hibernate 花时间按实体对插入进行分组，创建更大的批处理。

**因此，第二次运行测试时，我们会看到插入被分批:**
