接下来，我们需要为 feign 客户端设置日志级别。请注意，上一步只是为了启用日志记录。

有四种日志记录级别可供选择:

*   `NONE:`无日志记录(默认)
*   `BASIC:`记录请求方法和 URL 以及响应状态代码和执行时间
*   `HEADERS:`记录基本信息以及请求和响应头
*   `FULL`:记录请求和响应的头、主体和元数据

我们可以通过 java 配置或在我们的属性文件中配置它们。
