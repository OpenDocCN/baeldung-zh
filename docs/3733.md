## **1。概述**

在本文中，我们将看看`StreamUtils`类以及如何使用它。

简单地说，`StreamUtils`是一个 Spring 的类，包含一些处理流的实用方法——`InputStream`和`OutputStream`,它们驻留在包`java.io` 中，与 Java 8 的流 API 无关。
