## 2.比较数组

我们将比较 Java 中的数组，正如我们所知，这些是对象。因此，让我们刷新一些基本概念:

*   对象有引用和值
*   两个相等的引用应该指向相同的值
*   两个不同的值应该有不同的引用
*   两个相等的值不一定有相同的引用
*   原始值只能按值进行比较
*   字符串文字只能按值进行比较
