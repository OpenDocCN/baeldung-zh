## **2。`Stream.collect()`法**

`Stream.collect()`是 Java 8 的`Stream API`的终端方法之一。它允许我们执行可变的折叠操作(将元素重新打包到一些数据结构中，并应用一些额外的逻辑，将它们连接起来，等等)。)在一个`Stream`实例中保存的数据元素上。

该操作的策略通过`Collector`接口实现提供。
