## **1。概述**

Jersey 是一个用于开发 RESTFul Web 服务的开源框架。它还具有强大的内置客户端功能。

在这个快速教程中，我们将探索使用 [Jersey 2](https://web.archive.org/web/20221129003653/https://jersey.java.net/) 创建 JAX-RS 客户端。

关于使用 Jersey 创建 RESTful Web 服务的讨论，请参考本文。
