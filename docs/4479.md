# Baeldung 每周评论 27

> 原文：<https://web.archive.org/web/20220930061024/https://www.baeldung.com/2015-week-review-27>

2014 年初，我决定跟踪我的阅读习惯，并在 Baeldung 上分享最好的东西。

2014 年是不平凡的一年，每周都有回顾。我一直在做更多的阅读，以确保我涵盖和策划的东西有价值，实际上值得一读。

如果你觉得我的评论有趣且有用，请在评论中告诉我。

## **1。Spring 和 Java**

#### **[>>Jigsaw 项目带给 Java 9 的特性](https://web.archive.org/web/20220120150401/http://blog.codefx.org/java/dev/features-project-jigsaw-java-9/)**

#### **[> >谁在乎 toString 的表现？](https://web.archive.org/web/20220120150401/http://antoniogoncalves.org/2015/06/30/who-cares-about-tostring-performance/)**
