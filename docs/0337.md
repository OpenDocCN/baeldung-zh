# Baeldung 每周评论 29

> 原文：<https://web.archive.org/web/20220930061024/https://www.baeldung.com/2014-week-review-29>

在 2014 年初，我决定开始更好地记录我的阅读习惯，并在这里与大家分享。

重点是双重的——通过策展和记录，我的阅读变得更加有目的和多样化。此外，我相信好内容的监管会带来很多价值，帮助人们探索，让最好的东西上升到顶端。

希望您会喜欢我们在 2014 年下半年发布的这些内容。

**开始了……**

## **1。Java**

#### **[> >为数据访问代码编写测试——不要测试框架](https://web.archive.org/web/20220521215244/http://www.petrikainulainen.net/programming/testing/writing-tests-for-data-access-code-dont-test-the-framework/)**

让我们开始本周的回顾吧——“不要测试框架”的信息是你最重要的收获之一

#### **[> >如何通过 7 次 Logback 调整立即提高您的 Java 日志记录能力](https://web.archive.org/web/20220521215244/http://www.takipiblog.com/2014/07/23/how-to-instantly-improve-your-java-logging-with-7-logback-tweaks/)**

这篇文章填补了生态系统中的一个空白——以真实的数字为依据，仔细研究了当今伐木业中最常见的伐木附加物。

#### **[> >保持事物干燥:方法超载](https://web.archive.org/web/20220521215244/http://blog.jooq.org/2014/07/21/keeping-things-dry-method-overloading/)**

API 设计是细致入微的，你必须注意很多方面——这方面的好资源很少，所以这一篇非常值得一读。

#### **[> >冬眠隐藏宝石:pooled-lo 优化器](https://web.archive.org/web/20220521215244/http://vladmihalcea.com/2014/07/21/hibernate-hidden-gem-the-pooled-lo-optimizer/)**

本文继续这个系列，在标识符和生成策略上更进一步。从这篇文章中我学到了很多——我并不知道生成标识符的大多数策略，但是我肯定会更加关注我的标识符。

## **2。弹簧**

#### **[> > Spring 框架 4.1 发布候选可用](https://web.archive.org/web/20220521215244/https://spring.io/blog/2014/07/21/spring-framework-4-1-release-candidate-available)**

本周非常酷的消息——**Spring 4.1 第一个 RC 出来了**。跳上来，看看有什么新消息——这么多东西。

#### **[> > Spring 框架 4.1——处理静态 web 资源](https://web.archive.org/web/20220521215244/https://spring.io/blog/2014/07/24/spring-framework-4-1-handling-static-web-resources)**

用 Spring 为你的**静态资源服务并不难——但是像**一个很好的参考应用**和很好地使用现有工具这样的事情是可靠的，并且是受欢迎的改进。**

#### **[>>$ text $用 Spring Data 搜索你的文档 MongoDB](https://web.archive.org/web/20220521215244/https://spring.io/blog/2014/07/17/text-search-your-documents-with-spring-data-mongodb)**

使用 Mongo 和 Spring 数据进行全文搜索——看起来不用像 SOLR 这样成熟的解决方案也能做很多事情。

#### **[> >拖尾文件——Spring web socket 示例](https://web.archive.org/web/20220521215244/http://www.java-allandsundry.com/2014/07/tailing-file-spring-websocket-sample.html)**

这是 Spring 中对 **websocket 支持的一个很酷的应用——给文件加尾。**
