## 1.概观

[帕斯卡三角形](https://web.archive.org/web/20221218181728/https://en.wikipedia.org/wiki/Pascal%27s_triangle)是二项式系数以三角形形式的排列。帕斯卡三角形的数字是这样排列的，每个数字都是它上面两个数字的和。

在本教程中，我们将看到如何用 Java 打印帕斯卡的三角形。
