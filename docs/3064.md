## 1.概观

三元条件运算符 `?:`允许我们在 Java 中定义表达式。**它是也返回值的`if-else`语句的浓缩形式。**

在本教程中，我们将学习何时以及如何使用三元结构。我们将从查看它的语法开始，然后探究它的用法。
