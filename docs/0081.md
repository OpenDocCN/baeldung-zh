RxJava 是针对 Java 环境的[反应式扩展](https://web.archive.org/web/20220524025941/http://reactivex.io/)实现。

该库利用功能性和反应性技术的组合，可以代表事件驱动编程的一种优雅方法——值随时间变化，消费者在数据到来时做出反应。
