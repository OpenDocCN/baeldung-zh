## 1.概观

在这个快速教程中，我们将学习 Java 中的 [`Cipher`](/web/20220628062613/https://www.baeldung.com/java-cipher-class) 类。然后，我们将看到如何列出可用的密码算法及其提供商。

## 2.密码课

位于`javax.crypto`包中的`Cipher`类是 Java 加密扩展(JCE)框架的核心。这个框架提供了一组用于数据加密、解密和散列的加密密码。
