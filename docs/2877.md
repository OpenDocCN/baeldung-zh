## **1。概述**

在本教程中，我们将向**展示如何直接从 Java 代码**中运行 JUnit 测试——在一些场景中，这个选项会派上用场。

如果你是 JUnit 新手，或者如果你想升级到 JUnit 5，你可以查看一些我们关于这个主题的教程。
