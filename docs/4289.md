# Java 数据对象指南

> 原文：<https://web.archive.org/web/20220930061024/https://www.baeldung.com/jdo>

## **1。概述**

Java 数据对象是一个 API，设计用于将面向对象的数据保存到任何数据库中，并使用 Java 语法提供用户友好的查询语言。

在这篇文章中，我们将看到如何使用 JDO API 在数据库中持久化我们的对象。
