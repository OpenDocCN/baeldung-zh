#### **[> >真可惜 Java 8 没有 Iterable.stream()](https://web.archive.org/web/20220521213146/http://blog.jooq.org/2014/12/15/really-too-bad-that-java-8-doesnt-have-iterable-stream/)**

## **2。技术和思考**

#### **[>>API 不应受版权保护](https://web.archive.org/web/20220521213146/http://martinfowler.com/articles/copyright-api.html)**

这展示了**不让 API 受版权保护的背后的想法**。这是一篇重要的 5 分钟阅读材料——继续读下去。

#### **[> >在 Devs 中我们信任](https://web.archive.org/web/20220521213146/http://www.daedtech.com/in-devs-we-trust)**

敏捷的意识形态基础与行业成熟时完全不同。这并不奇怪，但是理解区别以及为什么“敏捷认证”是愚蠢的是很重要的。

#### **[> >黑客攻击重要](https://web.archive.org/web/20220521213146/http://randsinrepose.com/archives/hacking-is-important/)**

**“黑客”的风气**作为不前进而是跃进的力量。

#### **[> >大公司为何减速，又该如何应对](https://web.archive.org/web/20220521213146/http://dandreamsofcoding.com/2014/12/16/why-big-companies-slow-down/)**

一篇关于软件公司成长周期的长篇文章，非常有趣。

#### **[>>](https://web.archive.org/web/20220521213146/http://techblog.bozho.net/in-favour-of-self-signed-certificates/)**

围绕所有人的安全和 HTTPS 证书的有趣想法。

#### **[>>TDD](https://web.archive.org/web/20220521213146/http://blog.cleancoder.com/uncle-bob/2014/12/17/TheCyclesOfTDD.html)的周期**

**在纳米水平及以上做 TDD** 意味着什么。

#### **[>程序员知识](https://web.archive.org/web/20220521213146/http://henrikwarne.com/2014/12/15/programmer-knowledge/)**

程序员知识的半衰期就是为什么停止学习从来都不是一个好主意。
