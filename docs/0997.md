## **7。结论**

本教程讨论了如何为 Apache HttpClient 配置 SSL，以便它能够使用任何 HTTPS URL，而不管证书如何。也显示了弹簧`RestTemplate`的相同配置。

然而，需要理解的一件重要事情是**这种策略完全忽略了证书检查**——这使得它不安全，并且只在有意义的地方使用。

这些例子的实现可以在 GitHub 项目中找到——这是一个基于 Eclipse 的项目，所以它应该很容易导入和运行。
