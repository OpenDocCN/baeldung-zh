## 1.概观

[标准偏差](https://web.archive.org/web/20221208143845/https://en.wikipedia.org/wiki/Standard_deviation)(符号为 sigma–σ)是数据围绕平均值分布的度量。

在这个简短的教程中，我们将看到如何在 Java 中计算标准差。

## 2.计算标准偏差

标准差的计算公式是(∑(ų–Xi)^ ^ 2)/n 的平方根，其中:

*   ∑是每个元素的总和
*   Xi 是数组中的每个元素
*   ų是数组元素的平均值
*   n 是元素的数量
