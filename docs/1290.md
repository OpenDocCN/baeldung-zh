2014 年初，我决定跟踪我的阅读习惯，并在 Baeldung 上分享最好的东西。

2014 年是不平凡的一年，每周都有回顾。我一直在做更多的阅读，以确保我涵盖和策划的东西有价值，实际上值得一读。

如果你觉得我的评论有趣且有用，请在评论中告诉我。

**开始了……**

## **1。春天和 Java**

#### **[> > Java 版本统计:2015 版](https://web.archive.org/web/20220926181335/https://plumbr.eu/blog/java/java-version-statistics-2015-edition)**

深入了解 Java 8 在业界的采用情况。

#### **[> > Maven 集成测试在额外的源文件夹](https://web.archive.org/web/20220926181335/http://blog.code-cop.org/2015/04/maven-integration-tests-extra-folder.html)**

在 Maven 项目中分离集成测试的**的快速和要点指南。**

另一种避免使用额外插件的方法是简单地为测试定义命名模式，并基于正则表达式运行它们。

#### **[> >从战壕中跃出:返回 Git 提交信息为 JSON](https://web.archive.org/web/20220926181335/http://www.petrikainulainen.net/programming/spring-framework/spring-from-the-trenches-returning-git-commit-information-as-json/)**

详细、清晰地指导**发布关于已部署应用**的 Git 信息。这是那种你知道你不再是在做一个玩具应用的事情。

#### **[>>Spring Security 4.0:web socket，Spring 数据和测试支持](https://web.archive.org/web/20220926181335/http://www.infoq.com/news/2015/04/spring-security-4)**

Spring Security 4 新推出的**功能的简明摘要。**

#### **[> >春天:注入列表、地图、选项和 getBeansOfType()陷阱](https://web.archive.org/web/20220926181335/http://www.nurkiewicz.com/2015/04/spring-injecting-lists-maps-optionals.html)**

在春季连接更复杂的 beans 可能是一件棘手的事情——但肯定可以完成，而且做得很好。

## **3。沉思**

#### **[> >沼泽](https://web.archive.org/web/20220926181335/http://dandreamsofcoding.com/2015/04/21/the-swamp/)**

这是一个快速而简单的分析，讲述了在处理复杂、困难的问题时**过于乐观的常见问题**。

#### **[> >敏捷采用德雷福斯模式](https://web.archive.org/web/20220926181335/http://lizkeogh.com/2015/04/22/a-dreyfus-model-for-agile-adoption/)**

关于我们实践敏捷的成熟度的非常有趣的文章。

本周有数量惊人的好作品。

#### **[>定义企业层级](https://web.archive.org/web/20220926181335/http://www.daedtech.com/defining-the-corporate-hierarchy)**

对失败者/无能者/反社会者企业原型的一个有希望的重新定义是一个更温和但也许更微妙的拓扑结构。一本书也正在筹备中。

#### **[> >狂欢兑现:对资历的膜拜](https://web.archive.org/web/20220926181335/http://www.daedtech.com/carnival-cash-the-cult-of-seniority)**

一剂实用主义良药，也是工程师们开始工作的绝佳读物。

#### **[> >勇于创新，独自工作](https://web.archive.org/web/20220926181335/http://lemire.me/blog/archives/2015/04/20/to-be-creative-work-lone/)**

当你不是一个人的时候，很难——几乎不可能——进入心流状态；一个很好的例子。
