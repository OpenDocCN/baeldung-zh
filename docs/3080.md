本周，我选择了 [Datadog](/web/20221129021720/https://www.baeldung.com/datadog) ，这是我接受的 Java 周刊时事通讯的第一个赞助商。

六个月前，我试着发起了赞助活动，并拒绝了一些公司，因为各种原因(主要是因为我不相信他们的产品)。

我以前没有尝试过 Datadog，但是我已经使用了很多其他的 APM 解决方案，所以我知道会发生什么。我已经用他们的系统玩了一个星期了，我非常高兴他们成为我的第一个官方赞助商。

这是一个可靠的、超级成熟的解决方案，**它实际上从一开始就很有用，不需要我花一整天的时间来设置它**，也不需要花费太多。

所以，他们是我这周的选择。
