#### **[>>2016 年如何不讨厌春天](https://web.archive.org/web/20220813055325/https://spring.io/blog/2015/11/29/how-not-to-hate-spring-in-2016)**Spring . io

**[> >将 OAuth2 应用从 Spring Boot 1.2 迁移到 1.3](https://web.archive.org/web/20220813055325/https://spring.io/blog/2015/11/30/migrating-oauth2-apps-from-spring-boot-1-2-to-1-3)** [ spring.io ]

如果你正在做任何 OAuth2 工作——从我收到的关于它的问题数量来看—**越来越多的开发者正在利用 Spring Security OAuth**——这绝对是一篇你不想错过的文章。

#### **[>>Java 7 和 Java 8](https://web.archive.org/web/20220813055325/http://blog.jooq.org/2015/12/02/a-subtle-autocloseable-contract-change-between-java-7-and-java-8/)**【jooq.org】之间一个微妙的可自动关闭的契约变化

你每天都在学习新的东西——**`AutoClosable`实际上比 Java 7 中的**更加细致入微。

#### **[> >哪个线程执行 CompletableFuture 的任务和回调？](https://web.archive.org/web/20220813055325/http://www.nurkiewicz.com/2015/11/which-thread-executes.html)**【nurkiewicz.com】
