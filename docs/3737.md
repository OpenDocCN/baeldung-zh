## **1。概述**

在本教程中，我们将通过一个简单的场景来说明如何在 Spring Security 中使用 Run-As 身份验证。

关于 Run-As 的非常高级的解释如下:一个用户可以作为另一个拥有不同特权的主体执行一些逻辑。
