## 1.概观

在本教程中，我们将学习使用一个简单的 Java 程序来检查一个给定的字符串是否有效。一个 **[盘符](https://web.archive.org/web/20221208143832/https://en.wikipedia.org/wiki/Pangram)是任何一个包含给定字母表中所有字母至少一次的字符串。**
