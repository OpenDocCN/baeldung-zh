### **4.1。桥接方法**

为了解决上面的边缘情况，编译器有时会创建一个桥方法。这是 Java 编译器在编译扩展参数化类或实现参数化接口的类或接口时创建的合成方法，其中方法签名可能略有不同或不明确。
