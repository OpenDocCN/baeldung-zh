# Baeldung 每周评论 31

> 原文:[https://web . archive . org/web/20220930061024/https://www . bael dung . com/2014-week-review-31](https://web.archive.org/web/20220930061024/https://www.baeldung.com/2014-week-review-31)

在 2014 年初，我决定开始更好地记录我的阅读习惯，并在这里与大家分享。

重点是双重的——通过策展和记录，我的阅读变得更加有目的和多样化。此外，我相信好内容的监管会带来很多价值，帮助人们探索，让最好的东西上升到顶端。

希望您会喜欢我们在 2014 年下半年发布的这些内容。

**开始了……**

## **1。Java 和 Spring**

#### **[> >编写干净的测试——验证或不验证](https://web.archive.org/web/20220521212419/http://www.petrikainulainen.net/programming/testing/writing-clean-tests-to-verify-or-not-to-verify/)**

大多数文章使用简单的例子——表达观点，进入和退出。这不是这些文章中的一篇。编写真正的软件是一项复杂的工作，在这里——我们可以看到它的样子—**一个经过严格测试的实际用例单元**。然后，这些单元测试被重新评估和改进。

值得花时间读一下这篇文章。

#### **[>>JPA/Hibernate flush 攻略初级指南](https://web.archive.org/web/20220521212419/http://vladmihalcea.com/2014/08/07/a-beginners-guide-to-jpahibernate-flush-strategies/)**

概述了**休眠刷新机制**的工作原理——这是另一个增强您的休眠知识的工具。

#### **[> > Spring MVC 集成测试:断言给定的模型属性有全局错误](https://web.archive.org/web/20220521212419/http://blog.codeleak.pl/2014/08/spring-mvc-test-assert-given-model-attribute-global-errors.html)**

创建**一个自定义约束和自定义验证器**并在 Spring 控制器中使用它们的很酷的例子。

#### **[>>4 大 Java Web 框架揭晓:Spring MVC、Vaadin、GWT 和 JSF](https://web.archive.org/web/20220521212419/http://zeroturnaround.com/rebellabs/top-4-java-web-frameworks-revealed-real-life-usage-data-of-spring-mvc-vaadin-gwt-and-jsf/)** 的真实使用数据

关于框架在现实世界中使用的统计数据总是很好的——如果有 Spring 的加入——那就更好了。这里有些有趣的数字。

#### **[> > Spring Boot 管理](https://web.archive.org/web/20220521212419/https://blog.codecentric.de/en/2014/08/spring-boot-admin-2/)**

一个有趣的小启动应用程序，用于提供关于您部署的应用程序的快速反馈和信息。

#### **[> > CRUD 使用 Spring 数据 Rest，AngularJS 使用 Spring Boot](https://web.archive.org/web/20220521212419/http://www.programming-free.com/2014/07/spring-data-rest-with-angularjs-crud.html)**

这个应用中使用了很多很酷的技术——如果你想去 Spring Boot 或其他地方，一个示例应用是一个很好的起点。这篇文章也没有抓住你的手——它把你扔进了深渊，你不得不开始游泳。
