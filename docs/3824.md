[PRE1]

如上所示，我们使用`exec.mainClass `系统属性来传递完全限定的类名。

此外，我们必须确保在运行 main 方法之前，类路径已经准备好。这就是我们在执行 main 方法之前编译源代码的原因。

我们可以用普通的`java `和`javac. `实现同样的事情，但是，当我们使用一个相当大的类路径时，这可能会很麻烦。相反， **使用这个插件时，Maven 会自动负责填充类路径。**
