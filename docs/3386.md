## **1。概述**

在这个快速教程中，我们将使用 Spring Security 实现一个基本的解决方案来防止[暴力认证尝试](/web/20221005021715/https://www.baeldung.com/cs/brute-force-cybersecurity-string-search) 。

简而言之，我们将记录从单个 IP 地址发起的失败尝试的次数。如果该特定 IP 超过了设定的请求数量，它将被阻止 24 小时。
