# 用 Java 查找数组中的最小值/最大值

> 原文：<https://web.archive.org/web/20220930061024/https://www.baeldung.com/java-array-min-max>

## **1。简介**

在这个简短的教程中，我们将看到如何使用 Java 8 的`Stream` API 找到数组中的最大值和最小值。

我们将从寻找整数数组中的最小值开始，然后我们将寻找对象数组中的最大值。
